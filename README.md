# DayConverter

  - Simple service that convert day id to date (NEXT) in calendar  

  # Base URL :
   
              https://gad25.xyz/eduaction/d_converter.php

 # Parameters :
              - day_id (int)
                          1 -> Saturday
                          2 -> Sunday
                          3 -> Monday   
                          4 -> Tuesday
                          5 -> Wednesday
                          6 -> Thursday
                          7 -> Friday
                          other -> Invalid day id
